It sets wallpaper from bing wallpaper. Currently works for mac, have to extend it to linux.

* Step 1: Clone repo.

* Step 0: Create folder `~/Pictures/BingWallpapers/`. Currently path is hardcoded.

* Step 2: python bing_wallpaper.py
